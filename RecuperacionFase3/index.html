Propósito

La presente pagina web, fue diseñada con el proposito de informar, que son y como podemos utilizar las Etiquetas Semánticas y CSS Grid.

Introducción

Las etiquetas semánticas ayudan a definir la estructura del documento y permiten que las páginas web sean mejor indexadas por los buscadores. CSS Grid es un elemento que facilita el trabajo del desarrollador en front end, debido a su flexibilidad e implementación desde CSS sin la necesidad de realizar cambios en el HTML.
